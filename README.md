 Fitness AI Coach (LangChain + RAG)

An AI-powered fitness chatbot that delivers personalized workout and diet guidance using LangChain and Retrieval-Augmented Generation (RAG).

=>Features
- Chat-based fitness assistant
- PDF knowledge retrieval (RAG)
- Context-aware responses with memory
- Workout & diet recommendations
- Tech Stack

Python, Streamlit, LangChain, Groq (LLaMA 3), ChromaDB, HuggingFace Embeddings

▶️ Run
pip install -r requirements.txt
streamlit run app.py
💡 Example
"Weight loss plan"
"Beginner home workout"
"Muscle gain diet"
Overview:
Uses LangChain + RAG to retrieve relevant content from a fitness PDF and generate accurate, context-aware responses.

⭐ Simple AI fitness assistant using modern LLM architecture.
